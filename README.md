# Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.
This project helps beginners understand DOM manipulation, event handling, and game logic.

# feature

2-player gameplay (Player X and O)
Detects winners automatically
Highlights winning pattern (if added)
Reset game option
Clean UI with responsive design
Simple and beginner-friendly JavaScript logic

# Game Rules
The game is played on a 3×3 grid.
Player O starts the game.
Players take turns marking empty squares.
First player to get three marks in a row (horizontally, vertically, or diagonally) wins.
If all 9 squares are filled with no winner → It's a draw!
